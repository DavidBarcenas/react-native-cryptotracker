# Crypto Tracker

Application made with react-native to follow the crypto market

## Preview

![](/.readme-static/app.jpg)

## Project setup
```
npm install
```

### Install on device
```
npm run android/ios
```

### Compiles and hot-reloads for development
```
npm run start
```

### Run your tests
```
npm run test
```
